# coexistence-paper
Scripts and datasets to generate figures for coexistence paper. 

## Figures 1,2 and 5
- Run Figure1_mainR with the corresponding datasets (ancestor_radius.txt, large_radius.txt, small_radius.txt and panel-c-aspect-ratio.R) to generate Figure 1 panels.
- Run Figure2.m with the corresponding dataset (rawdt_for_plot_interpolationNAs_filtered.txt) to generate Figure 2.
0 Run Figure5_main.R with the corresponding datasets (see raw data excel file) to generate Figure 5.

## Figure 3 and 4
- Run Figure3_create_figure_coexistence_zones.m to generate Figure 3.
- Run Figure4_stepbystepinvasion.m to generate Figure 4.
